```qml
readonly property int radiusSm: 6
```

```qml
readonly property int spacingXS: 4

```

## 使用指南

### 圆角
- **小圆角 (6px)**：按钮、输入框、小卡片
- **中圆角 (10px)**：卡片、面板、弹窗
- **大圆角 (14px)**：大型对话框
- **胶囊形 (999px)**：圆形按钮、标签

### 间距
- **4px (XS)**：图标与文字间距
- **8px (S)**：按钮内边距、列表项间距
- **12px (M)**：卡片内边距
- **16px (L)**：区块间距
- **24px (XL)**：大区块间距
