# 输入框组件

## TextField

```
普通状态：
┌─────────────────────────┐
│ 请输入文本...            │
└─────────────────────────┘

聚焦状态：
┏━━━━━━━━━━━━━━━━━━━━━━━━━┓
┃ 输入的文本|              ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━┛
  (活动色边框，光标闪烁)
```

- 用途：用于单行文本输入，如名称、路径、关键词和表单字段。
- 结构：由输入区、占位文本、焦点边框和可选前后缀元素组成。
- 视觉：高度与按钮、下拉框等基础控件保持统一；圆角使用 `Theme.radiusSm`。
- 交互：支持焦点、选中、错误、禁用和只读状态；焦点态优先通过边框和光标表达。

## SearchField

```
┌─────────────────────────┐
│ 🔍 搜索...            ✕ │
└─────────────────────────┘
  图标   占位符      清除
```

- 用途：用于搜索、过滤和快速定位内容。
- 结构：通常包含搜索图标、输入区和可选清空按钮。
- 行为：支持输入即过滤、`Enter` 触发搜索和一键清除；清空后应保持焦点，方便继续输入。
