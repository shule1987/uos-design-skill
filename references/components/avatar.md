# 头像组件

## Avatar
- 用途：用于展示用户、成员或对象的身份标识。
- 内容来源：优先显示图片；图片缺失时退回姓名首字母、缩写或默认占位图形。
- 视觉：头像通常保持正圆形，尺寸跟随场景变化；常见于列表、评论区、成员面板和标题区。
- 交互：如果头像可点击，应提供明确 hover 与焦点态；如果只是展示元素，不要让它抢占交互焦点。

## AvatarGroup
- 用途：用于同时展示多个成员或协作者。
- 布局：多个头像可使用轻度重叠排列，保证头像之间仍可辨认。
- 截断：当成员数量过多时，保留前几项并使用 `+N` 表示剩余人数。
