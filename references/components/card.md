# 卡片组件

## Card

```
┌──────────────────────┐
│                      │
│    [缩略图区域]       │
│     (200x160)        │
│                      │
├──────────────────────┤
│  卡片标题             │
│                      │
│  这里是卡片的描述     │
│  文本内容...          │
│                      │
└──────────────────────┘
   (280x320, 带阴影)
```

- 用途：用于承载一组相对独立的信息块，如文档、素材、结果项和推荐内容。
- 结构：通常由缩略图、标题、描述和可选操作组成。
- 视觉：容器圆角使用 `Theme.radiusMd`，背景使用 `Theme.cardBg` 或相近表面色，阴影保持轻量。
- 交互：整张卡片可点击时，要和局部操作按钮区分交互热区，避免误触。
