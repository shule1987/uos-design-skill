# 卡片组件

## Card

```
┌──────────────────────┐
│                      │
│    [缩略图区域]       │
│     (200x160)        │
│                      │
├──────────────────────┤
│  卡片标题             │
│                      │
│  这里是卡片的描述     │
│  文本内容...          │
│                      │
└──────────────────────┘
   (280x320, 带阴影)
```

> 示例描述：这里定义 `Card` 组件，基于 `Rectangle` 实现，对外暴露 `title`、`description` 和 `thumbnail` 等属性。 尺寸与样式上宽度使用 `280`，高度使用 `320`，圆角使用 `Theme.radiusMd`；结构上使用 `Column` 组织内容；交互上覆盖悬停反馈、过渡动画。
