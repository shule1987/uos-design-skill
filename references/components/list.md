# 列表组件

## ListItem
- 用途：用于导航列表、文件列表、设置项列表和结果列表。
- 结构：可包含图标、标题、副标题、状态文本、尾部操作或选中指示。
- 尺寸：单行信息可采用更紧凑高度，双行信息使用更高行高并增加上下留白。
- 交互：支持 hover、选中、点击、键盘导航和无障碍名称；当前项不应只靠文字加粗表达。

## Trailing Control Rows

- 当列表项右侧承载 `Switch`、`CheckBox`、`ComboBox`、按钮组或其他尾部操作时，应把右侧控件放进独立 trailing slot，而不是直接跟在文本列后面。
- 在同一组重复列表项中，所有 trailing slot 必须共享统一的右对齐终点。
- 标题长短、副标题有无、状态徽标、禁用态和当前值文本都不能改变 trailing slot 的最终右边界。
- 优先复用统一的双槽行组件，例如“左信息列 + 右控件槽”的 `SettingRow` 风格原语。
