# 进度条组件

## ProgressBar

```
0%:  ┌────────────────────┐
     └────────────────────┘

50%: ┌──────────┬─────────┐
     │██████████│         │
     └──────────┴─────────┘

100%:┌────────────────────┐
     │████████████████████│
     └────────────────────┘
```

- 用途：用于展示线性进度，如下载、导入、安装和同步任务。
- 规则：当任务可量化时显示明确进度；不可量化时改用 `BusyIndicator`。
- 视觉：已完成区和未完成区的对比要清晰，但不应抢走主要内容注意力。

## CircularProgress

```
    ╭─────╮
   ╱   75% ╲
  │    ●    │
   ╲       ╱
    ╰─────╯
   (圆形进度)
```

- 用途：用于仪表盘、状态卡片或局部任务进度。
- 规则：适合承载单个关键数值，不适合展示过多同时进行的任务。

## BusyIndicator
- 用途：用于不可量化或短时加载中的等待反馈。
- 行为：只要任务仍在进行，就保持稳定动画；结束后应及时隐藏，避免残留误导。
