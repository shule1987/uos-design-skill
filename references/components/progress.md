# 进度条组件

## ProgressBar

```
0%:  ┌────────────────────┐
     └────────────────────┘

50%: ┌──────────┬─────────┐
     │██████████│         │
     └──────────┴─────────┘

100%:┌────────────────────┐
     │████████████████████│
     └────────────────────┘
```

## CircularProgress

```
    ╭─────╮
   ╱   75% ╲
  │    ●    │
   ╲       ╱
    ╰─────╯
   (圆形进度)
```

> 示例描述：这里定义 `ProgressBar` 组件，基于 `Rectangle` 实现，对外暴露 `value` 等属性。 尺寸与样式上宽度使用 `200`，高度使用 `4`，圆角使用 `2`；结构上使用 `Rectangle` 组织内容；交互上覆盖过渡动画。

## CircularProgress
> 示例描述：这里定义 `CircularProgress` 组件，基于 `Item` 实现，对外暴露 `value` 和 `size` 等属性。 尺寸与样式上宽度使用 `size`，高度使用 `size`；结构上使用 `Canvas` 组织内容。

## BusyIndicator
> 示例描述：这里定义 `BusyIndicator` 组件，基于 `Item` 实现，对外暴露 `size` 和 `running` 等属性。 尺寸与样式上宽度使用 `size`，高度使用 `size`，透明度使用 `running ? 1 : 0`；结构上使用 `AppIcon` 组织内容；交互上覆盖过渡动画。
