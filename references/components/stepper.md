# 步骤条组件

## Stepper

```
  ①────────②────────③────────④
 完成     进行中    待处理    待处理
 ✓        2        3        4
[绿色]   [活动色] [灰色]   [灰色]

步骤1    步骤2    步骤3    步骤4
已完成   进行中   等待中   等待中
```

> 示例描述：这里定义 `Stepper` 组件，基于 `Row` 实现，对外暴露 `steps` 和 `current` 等属性。 尺寸与样式上间距使用 `0`；结构上使用 `Repeater` 组织内容；交互上覆盖过渡动画。
