# 步骤条组件

## Stepper

```
  ①────────②────────③────────④
 完成     进行中    待处理    待处理
 ✓        2        3        4
[绿色]   [活动色] [灰色]   [灰色]

步骤1    步骤2    步骤3    步骤4
已完成   进行中   等待中   等待中
```

- 用途：用于展示多步骤流程的当前位置和整体进度。
- 结构：由步骤节点、连接线和步骤标题组成，必要时可增加副标题或状态说明。
- 状态：至少区分已完成、进行中、待处理；必要时增加错误或阻塞态。
- 规则：步骤条主要用于表达流程阶段，不适合承载复杂分支逻辑。
