# 标签页组件

- 优先使用 DTK 标签栏；自定义标签页主要用于浏览器式或文档式主窗口。

## TabItem

```
标签栏示例：
┌────────┬────────┬────────┬───┐
│ 标签1  │ 标签2  │ 标签3  │ + │
│  [✕]   │  [✕]   │  [✕]   │   │
└────────┴────────┴────────┴───┘
  激活      普通      普通    新建

固定标签 (pinned):
┌───┬────────┬────────┐
│ 📌│ 标签1  │ 标签2  │
└───┴────────┴────────┘
48px   180px    180px
```

- 用途：用于并列承载多个文档、页面或工作区。
- 状态：至少区分激活、普通、悬停和固定标签；可选支持未保存、加载中等附加状态。
- 尺寸：固定标签可更紧凑，普通标签由标题长度和关闭按钮共同决定宽度。
- 交互：支持点击切换、关闭、键盘导航和拖拽重排；当前标签要比普通标签有更明确的背景或边界差异。
