# 标签页组件

- 优先使用 DTK 标签栏；自定义标签页主要用于浏览器式或文档式主窗口。

## TabItem

```
标签栏示例：
┌────────┬────────┬────────┬───┐
│ 标签1  │ 标签2  │ 标签3  │ + │
│  [✕]   │  [✕]   │  [✕]   │   │
└────────┴────────┴────────┴───┘
  激活      普通      普通    新建

固定标签 (pinned):
┌───┬────────┬────────┐
│ 📌│ 标签1  │ 标签2  │
└───┴────────┴────────┘
48px   180px    180px
```

> 示例描述：这里定义 `TabItem` 组件，基于 `Rectangle` 实现，对外暴露 `title`、`active` 和 `pinned` 等属性，并通过 `clicked` 和 `closeRequested` 发出交互事件。 尺寸与样式上宽度使用 `pinned ? 48 : 180`，高度使用 `30`，圆角使用 `Theme.radiusMd`；结构上使用 `Row` 和 `IconButton` 组织内容；交互上覆盖悬停反馈、点击触发、键盘回车和空格操作、无障碍语义、过渡动画。
