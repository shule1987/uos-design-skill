# 表格组件

## Table

```
┌─────────┬──────────┬──────────┬─────────┐
│ 姓名    │ 年龄     │ 邮箱     │ 操作    │
├─────────┼──────────┼──────────┼─────────┤
│ 张三    │ 25       │ zhang@   │ [编辑]  │
├─────────┼──────────┼──────────┼─────────┤
│ 李四    │ 30       │ li@      │ [编辑]  │
├─────────┼──────────┼──────────┼─────────┤
│ 王五    │ 28       │ wang@    │ [编辑]  │
└─────────┴──────────┴──────────┴─────────┘
```

> 示例描述：这里定义 `Table` 组件，基于 `Item` 实现，对外暴露 `columns` 和 `dataSource` 等属性。 结构上使用 `Column` 组织内容；交互上覆盖悬停反馈。
