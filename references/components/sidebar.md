# 侧边栏组件

## Sidebar
- 用途：用于一级导航、资源分组、筛选入口和少量全局操作。
- 结构：通常由顶部品牌区或标题区、导航项列表、可选的折叠按钮组成；如果存在折叠按钮，其位置应在 logo 右侧 `20px` 处。
- 尺寸与状态：展开宽度通常在 `200-240px`，并可按内容和窗口宽度调节；最小可调宽度为 `100px`。不要把 `60px` 图标栏当作默认或唯一折叠目标。
- 视觉：背景优先使用 `Theme.bgPanel`；在性能允许且平台支持时可增加轻度 blur，但必须有纯色回退。
- 交互：支持 hover、选中、键盘导航和无障碍名称；当导航条目较多时建议支持折叠，但不是所有侧边栏都必须强制提供。若支持折叠，则在宽度缩到 `100px` 或主动触发折叠按钮时应隐藏侧边栏，并以短时动画完成展开与收起。
