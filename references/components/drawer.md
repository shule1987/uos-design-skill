# 抽屉组件

## Drawer

```
右侧抽屉 (placement="right")
┌─────────────────────┬──────────┐
│                     │          │
│                     │  抽屉    │
│    主内容区域        │  内容    │
│                     │ (360px)  │
│                     │          │
└─────────────────────┴──────────┘

左侧抽屉 (placement="left")
┌──────────┬─────────────────────┐
│          │                     │
│  抽屉    │                     │
│  内容    │    主内容区域        │
│ (360px)  │                     │
│          │                     │
└──────────┴─────────────────────┘
```

- 用途：用于承载比 Popup 更复杂、但又不适合打断为模态对话框的补充内容。
- 位置：通常从左侧或右侧滑入；只有在场景明确时才使用底部抽屉。
- 尺寸：宽度根据内容复杂度决定，常见桌面场景可控制在中等固定宽度范围，不要吞掉整个主内容区。
- 交互：支持 `Esc` 关闭、遮罩点击关闭和明确的返回路径；打开与关闭都应使用简洁的位移过渡。
