# 抽屉组件

## Drawer

```
右侧抽屉 (placement="right")
┌─────────────────────┬──────────┐
│                     │          │
│                     │  抽屉    │
│    主内容区域        │  内容    │
│                     │ (360px)  │
│                     │          │
└─────────────────────┴──────────┘

左侧抽屉 (placement="left")
┌──────────┬─────────────────────┐
│          │                     │
│  抽屉    │                     │
│  内容    │    主内容区域        │
│ (360px)  │                     │
│          │                     │
└──────────┴─────────────────────┘
```

> 示例描述：这里定义 `Drawer` 组件，基于 `Item` 实现，对外暴露 `open`、`placement`、`size`、`hiddenX` 和 `hiddenY` 等属性。 结构上使用 `Rectangle` 组织内容；交互上覆盖点击触发、无障碍语义、过渡动画。
